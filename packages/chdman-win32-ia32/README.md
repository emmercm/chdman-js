# chdman-win32-ia32

The last version of `chdman.exe` binary that exists here is v0.286. This directory will remain as an archive for anyone seeking to download a specific version of the tool.

MSYS2/pacboy started dropping support for 32-bit packages in [December 2023](https://www.msys2.org/news/#2023-12-13-starting-to-drop-some-32-bit-packages); and v0.286 was the last version before it dropped support for 32-bit clang, libc++, lld, and llvm.

MAME hasn't had an official 32-bit release since [v0.217 (December 2019)](https://www.mamedev.org/oldrel.html).
